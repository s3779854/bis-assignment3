Project Progress: Testing

Testing was mostly done by the person who implemented the feature. If we were to do anything differently we would try to allocate time for more thorough testing as we had a few bugs that slipped through that could have been avoided, if we had dedicated time to testing for bug free implementation we would have been able to have a more polished experience for the player as well as further develop our knowledge of Unity.
